# flight-load-trim-app
A responsive web-based Load & Trim Calculator designed for quick aircraft weight and balance calculations.
The application allows users to enter passenger, baggage, and fuel data to calculate loading conditions and trim values efficiently.

Features
1. Aircraft load and trim calculations
2. Responsive design for desktop and mobile
3. Simple and user-friendly interface
4. Fast real-time calculations
5. Web app support
6. Offline-friendly interface
   
Technologies Used
HTML5
CSS3
JavaScript
